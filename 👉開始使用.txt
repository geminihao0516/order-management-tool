✅ 程式已完全修正，支援你的資料格式！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 第一步：啟動程式

   雙擊 → 啟動GUI.command

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 第二步：準備訂單資料（兩種方式）

   方式 A：從 Excel 複製貼上（推薦）
   ┌────────────────────────────────────┐
   │ 1. 在 Excel 選取資料（不含標題）  │
   │ 2. 複製 (Ctrl+C 或 Cmd+C)         │
   │ 3. 貼到「訂單資料輸入」區          │
   │                                    │
   │ ✅ Excel 會自動使用 Tab 分隔！    │
   └────────────────────────────────────┘

   方式 B：多行格式 + 轉換
   ┌────────────────────────────────────┐
   │ 1. 貼上多行格式資料                │
   │ 2. 點擊「🔄 轉換多行格式」        │
   │                                    │
   │ ✅ 自動轉換為正確格式              │
   └────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 資料格式要求（在 Excel 中）

   建立4欄表格：
   ┌───────────────────────────────────────┐
   │ 品項     │ 姓名/生日 │ 對象/生日 │ 願望 │
   ├───────────────────────────────────────┤
   │ 鬼王x2   │ 王小明/... │ 李美麗/... │ 事業順利 │
   │ 愛神x3   │ 陳大華/... │ —         │ 求好姻緣 │
   └───────────────────────────────────────┘

   重要：
   • 品項格式：名稱x數量（例如：鬼王x2）
   • 用 x 不是 *
   • 多個品項用 + 連接（例如：鬼王x2+三鬼頭x4）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 第三步：複製資料到 GUI

   1. 在 Excel 中選取資料（不含標題）
   2. 按 Ctrl+C (Win) 或 Cmd+C (Mac)
   3. 在 GUI 的「訂單資料輸入」區點擊
   4. 按 Ctrl+V (Win) 或 Cmd+V (Mac) 貼上

   ✅ Excel 會自動使用 Tab 分隔！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔍 第四步：貼上參考數據（選填）

   在「參考數據」區域貼上你的參考清單：

   87支鬼王
   101支三鬼頭
   24支直立大鬼
   19支大鬼頭
   3支愛神
   7支大鬼鎖心
   13支拆散
   7支依霸
   1支雙色直立燕通

   ✅ 每行一個品項，格式：數量+支+品項名稱

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 第五步：生成報表

   點擊 → 🚀 生成報表

   程式會自動：
   • 展開所有品項
   • 生成雙欄明細表
   • 統計各品項數量
   • 比對參考數據（如果有提供）
   • 檢測異常訂單

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💾 第六步：儲存或複製報表

   方法 A：儲存為檔案
   → 點擊「💾 儲存報表」
   → 選擇儲存位置
   → 完成

   方法 B：複製到剪貼簿
   → 點擊「📋 複製到剪貼簿」
   → 貼到 Word 或其他程式

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🧪 快速測試（建議先做）

   1. 點擊「📋 貼上範例資料」
   2. 點擊「📋 貼上範例參考數據」
   3. 點擊「🚀 生成報表」
   4. 查看結果

   ✅ 如果能成功生成，你就知道如何使用了！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 報表內容說明

   生成的報表包含：

   1️⃣ 報表摘要
      • 總訂單數
      • 總品項數
      • 品項種類數
      • 異常訂單數

   2️⃣ A4雙欄明細表
      • 左欄：編號 1 ~ N/2
      • 右欄：編號 N/2+1 ~ N
      • 每一支品項都有獨立編號

   3️⃣ 品項統計總表
      • 所有品項及數量
      • 總計

   4️⃣ 數量差異比對表
      • 系統統計 vs 參考數據
      • 標示差異

   5️⃣ 異常訂單明細
      • 重複訂購品項的訂單
